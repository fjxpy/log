# 分页组件 

## 属性 MyPager

| 属性名        | 含义         | 类型   | 必填 | 默认值 |
| :------------ | ------------ | ------ | ---- | ------ |
| total         | 页码总数     | Number | 否   | 0      |
| limit         | 每页显示条数 | Number | 否   | 10     |
| nowPage       | 当前页码数   | Number | 是   | 1      |
| visibleNumber | 可见的页码数 | Number | 否   | 10     |

##  事件

| 事件名     | 含义     | 事件参数 | 参数类型 |
| ---------- | -------- | -------- | -------- |
| pageChange | 页码变化 | newPage  | number   |

# 菜单组件

##  组件名 Mymenu

## 属性

| 属性名  | 含义     | 类型   | 必填 | 默认值 |
| ------- | -------- | ------ | ---- | ------ |
| memuArr | 菜单数组 | Arrary | 是   | []     |

```js	
const insData= [{
    id,
    parent_id,
    children_len,
    router_name,
    href,
    menu,
    type,
    level,
    key,
    activeItem
}]
```
